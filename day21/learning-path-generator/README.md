# Learning Path Generator - Day 21

An AI-powered learning path generation system that creates personalized curriculum sequences using advanced algorithms and collaborative filtering.

## 🚀 Quick Start

1. **Start the system:**
   ```bash
   ./start.sh
   ```

2. **Access the application:**
   - Frontend Dashboard: http://localhost:3000
   - Backend API: http://localhost:8000
   - API Documentation: http://localhost:8000/docs

3. **Stop the system:**
   ```bash
   ./stop.sh
   ```

## 🧪 Testing

Run all tests:
```bash
./run_tests.sh
```

## 📋 Features

- **Intelligent Path Generation**: Uses topological sorting and difficulty progression
- **Personalization**: Adapts to user progress and learning patterns
- **Collaborative Filtering**: Leverages similar user patterns
- **Real-time Progress**: Live updates and recommendations
- **Modern Dashboard**: React-based UI with Material-UI components
- **Production Ready**: Docker support and comprehensive testing

## 🏗️ Architecture

- **Backend**: FastAPI with SQLAlchemy
- **Frontend**: React with Material-UI
- **Database**: PostgreSQL with Redis caching
- **AI Integration**: Gemini API ready
- **Testing**: Pytest and Jest

## 📊 Key Algorithms

1. **Prerequisite Resolution**: Modified topological sorting
2. **Difficulty Progression**: Adaptive curve generation
3. **Collaborative Filtering**: User similarity clustering
4. **Path Optimization**: Multi-objective optimization
